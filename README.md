# lab-hthaddon

本仓库用于归档和发布多个独立 HTML 页面。这些页面彼此独立，不针对任何特定域名或网站。

## 项目简介

这是一个轻量级的 HTML 页面集合仓库，主要用于存放各种独立的、功能明确的 HTML 文件。每个页面均可单独运行和访问，适合用于快速原型展示、文档说明或小型工具页面的托管。

## 目录说明

- `./` — 仓库根目录，直接存放各 HTML 文件及其相关资源
- 各页面文件以有意义的文件名命名，便于识别
- 资源文件（如 CSS、JS、图片等）一般与 HTML 文件放在同级目录或子目录中

## 页面归档说明

- 每个 HTML 页面均为独立项目，互不依赖
- 页面内容可能包含演示、说明或工具性质的功能
- 归档时尽量保持文件结构简洁，避免冗余嵌套
- 部分页面可能附带独立的样式表和脚本文件

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来改进现有页面或添加新的独立页面
- 新增页面请保持文件结构清晰，避免与已有页面产生冲突
- 不建议在页面中嵌入外部跟踪、广告或第三方服务
- 所有提交内容应遵守开源社区的基本规范

## 使用方式

可直接通过浏览器打开仓库中的任意 HTML 文件查看其内容。如需部署，可将整个仓库或所需页面复制到任意静态服务器中。
